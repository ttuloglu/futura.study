# Onboarding medya dosyalari

Bu klasorde onboarding su dosyalari beklenir:

- `fortale-onboarding-intro.mp4` (olusturuldu)
- `fortale-1.webp`
- `fortale-2.webp`
- `fortale-3.webp`
- `fortale-4.webp`
- `fortale-5.webp`
- `fortale-6.webp`

6 gorselin kaynak dosyalarini `public/onboarding/raw/` altina koyup su komutu calistir:

```bash
npm run optimize:onboarding
```

Komut:

- videoyu mobil dikey formata optimize eder
- gorselleri `fortale-1.webp` ... `fortale-6.webp` olarak daha dusuk boyutta uretir
